# Frontend Runbooks

Use this folder for actionable runbooks (deployments, environment fixes, incident response).

## Contribution Checklist

1. Start from `docs/templates/runbook-template.md`.
2. Include clear Trigger, Preconditions, Step-by-step, Verification, and Rollback sections.
3. Reference any tasks you create in `docs/tasks/backlog.md`.

## Candidate Runbooks

- Re-deploy frontend to Firebase Hosting
- Re-link Google OAuth credentials after rotation
- Diagnose Cloud Logging gaps from the browser
