# Worker Runbooks

Actionable guides for running and repairing the Python worker (Portainer, scheduler fixes, queue unstick).

## Contribution Checklist

1. Start from `docs/templates/runbook-template.md`.
2. Include rollback steps and verification checklists.
3. Reference backlog IDs for any long-term follow-ups.

## Candidate Runbooks

- Restarting the worker via Portainer
- Purging duplicate queue items
- Regenerating scheduler configs from scratch
