# Environment Variable Reference

This document consolidates the environment variables previously tracked in `env.example` and `.env.backup`. Replace secrets through your secure secret management workflow; do not commit plaintext secrets.

## GitHub Configuration

- `GITHUB_OWNER`
- `GITHUB_REPO`
- `GITHUB_TOKEN`

## Project Management

- `PM_EMAIL`
- `PROJECT_NAME`
- `WORKER_A_EMAIL`
- `WORKER_B_EMAIL`

- `FIREBASE_CLIENT_EMAIL`
- `FIREBASE_CLIENT_ID`
- `FIREBASE_CLIENT_X509_CERT_URL`
- `FIREBASE_PRIVATE_KEY`
- `FIREBASE_PRIVATE_KEY_ID`

- `FIREBASE_TOKEN_URI`
